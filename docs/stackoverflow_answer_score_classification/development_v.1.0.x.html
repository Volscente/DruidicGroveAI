Data Preparation

Transformations

  • Encode user_about_me in user_about_me_encoded
  • Extract year from user_creation_date in user_creation_year
  • Encode user_location in user_location_encoded
  • Standardise and remove outliers from: user_reputation, user_up_votes, user_down_votes, user_views
  • Engineer the flag user_has_profile_image_url when the user has the user_profile_image_url filled
  • Engineer the flag user_has_website_url when the user has the user_website_url filled
  • Compute the value user_badge_count as the amount of badges earned by the user
  • Engineer has_most_rare_badges if the user has one of the most rare badges from Table curated_stackoverflow_data_model.top_rarest_badges
  • Encode the answer_body in answer_body_encoded
  • Extract year from answer_creation_date in answer_creation_year
  • Extract month from answer_creation_date in answer_creation_month
  • Encode badge_names in badges_encoded
